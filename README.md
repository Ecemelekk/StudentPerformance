# StudentPerformance
Bu proje, öğrencilerin akademik başarılarını etkileyen sosyo-ekonomik ve davranışsal faktörleri analiz etmek amacıyla geliştirilmiş bir Eğitim Verisi Analizi çalışmasıdır. 12 yıllık eğitim teknolojileri deneyimimle, eğitimdeki "başarıyı getiren faktörleri" veriye dayalı olarak modellemeyi hedefledim.
